# Secure-Drone-Trajectory-Planning

Autonomous drone trajectory planning in known and unknown environments with decentralized security integration.

## 🚀 Project Overview
This repository implements an advanced autonomous navigation system for multi-drone swarms. It leverages **Deep Reinforcement Learning (PPO/SAC)** for continuous aerodynamic control and a **six-layer Blockchain architecture** for secure coordination and data integrity.

### 🛠️ Technical Core
- **AI Brain:** Proximal Policy Optimization (PPO) & Soft Actor-Critic (SAC).
- **Blockchain Consensus:** Credit-score and Grouping-mechanism Practical Byzantine Fault Tolerance (**CG-PBFT**).
- **Cryptography:** **ASCON** lightweight ciphers for energy-efficient encryption.
- **Data Structures:** 3D Segment Trees for optimized spatial range querying and Prioritized Experience Replay (PER).
- **Hardware Target:** Qualcomm Flight RB5 / RB6 robotics platforms.
